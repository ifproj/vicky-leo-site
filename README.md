# Vicky Leo — website

A static personal site (no build step). Open `index.html`, or serve the folder.
Content is editable via `admin.html` (passphrase is set in `assets/js/auth.js`).

This deployment is set to **noindex** (a review copy). To allow search engines
once approved, remove the `<meta name="robots" content="noindex">` line in
`index.html`.
